This replaces the technical agent builder as the default entry point for new users.

Database Schema (backend/prisma/schema.prisma):
- Add `BusinessProfile` model: id, organizationId, userId, businessName, industry (e-commerce|saas|real-estate|restaurant|healthcare|fitness|legal|education|agency|retail|other), description, websiteUrl, services (String[]), businessHours (JSON), contactInfo (JSON: { phone, email, address }), tone (friendly|professional|casual|formal|luxury), languages (String[]), primaryGoal (lead_gen|sales|support|bookings|orders|faq), avgDealValue, targetAudience, competitors (String[]), faqItems (JSON[]), createdAt, updatedAt
- Add `AgentCreationSession` model: id, organizationId, userId, businessProfileId, status (CHATTING|GENERATING|REVIEWING|DEPLOYED), messages (JSON[]: [{ role, content, type: text|suggestion|action }]), generatedAgentConfig (JSON), suggestedPlatforms (String[]), suggestedTools (String[]), websiteScrapedData (JSON), estimatedMonthlyVolume, createdAt, updatedAt

Backend API (backend/src/api/business-setup/routes.js):
- POST /api/business-setup/start — creates BusinessProfile + AgentCreationSession, returns sessionId + first AI question
- POST /api/business-setup/sessions/:id/chat — user sends message, AI responds with next question or summary. Stores conversation history.
- POST /api/business-setup/sessions/:id/generate — triggers AI backend to generate complete agent config from session messages + business profile
- GET /api/business-setup/sessions/:id/preview — returns generated config (name, prompt, greeting, platforms, tools, sample conversation)
- POST /api/business-setup/sessions/:id/deploy — one-click creates Agent, connects platforms, enables KB, returns live webhook URLs
- POST /api/business-setup/sessions/:id/scrape-website — accepts websiteUrl, fetches and extracts: business name, services, contact info, FAQ items, business hours. Stores in BusinessProfile.
- GET /api/business-setup/profiles/:id — get business profile
- PATCH /api/business-setup/profiles/:id — update business profile (simple fields only)

Backend Services:
- Create backend/src/services/business-setup-engine.js:
  - processChatMessage(sessionId, userMessage): maintains conversation state, asks smart follow-up questions based on industry. If user says "I run a dental clinic," next questions auto-focus on appointment booking, insurance, emergency protocol. If user says "I sell shoes online," questions focus on catalog, shipping, returns, payments.
  - generateAgentConfig(sessionId): compiles all answers into a complete Agent object:
    - name: "{BusinessName} AI Assistant"
    - systemPrompt: AI-generated from business description + goals + tone + industry best practices
    - greeting: contextual first message ("Hi! Welcome to {BusinessName}. I can help you book an appointment or answer questions about our services.")
    - persona: { tone, style, language }
    - capabilities: auto-selected based on industry + primaryGoal (e.g., restaurant → [appointment_booking, menu_display, order_taking, payment_link])
    - platforms: auto-selected based on industry (B2C local → WhatsApp + Instagram; B2B → Webchat; professional services → Webchat + WhatsApp)
    - tools: mapped from capabilities
    - knowledgeBaseEnabled: true (always on for business-setup agents)
  - scrapeWebsite(url): uses fetch + cheerio/jsdom to extract text, then calls AI backend to parse structured business data
  - createAgentFromSession(sessionId): creates Agent record, creates KnowledgeBase, associates them, returns agent

- Add to scheduler: weekly auto-health check for QuickAgents — if agent hasn't had conversations in 7 days, suggest re-engagement playbook

AI Backend (aibackend/app/routes.py):

Smart Question Flow (examples by industry):
- E-commerce: "What do you sell?" → "What's your price range?" → "Do you offer cash on delivery?" → "What's your return policy?" → "Do you ship nationwide?"
- Restaurant: "What's your cuisine?" → "Do you take reservations?" → "Do you offer delivery?" → "What are your peak hours?" → "Any dietary options?"
- Real Estate: "What locations do you cover?" → "Residential or commercial?" → "Do you do rentals or sales?" → "What's your commission structure?"
- SaaS: "What problem do you solve?" → "Who's your ideal customer?" → "Do you offer demos?" → "What's your pricing model?" → "Any free trial?"
- Healthcare: "What services do you offer?" → "Do you accept insurance?" → "Do you require appointments?" → "Emergency contact?" (adds HIPAA disclaimer auto)
